## 🔬 Flagship Projects

### [KAIROS](https://github.com/Pankaj4152/KAIROS) — Personal AI Assistant with Persistent Memory
A Jarvis-style assistant: voice at home, Telegram outside, browser UI everywhere. 3-tier LLM routing (local → cloud), hybrid SQLite + vector memory, async orchestration pipeline, resilient fallback system, proactive scheduling.
`Python · FastAPI · LiteLLM · Ollama · sqlite-vec · Docker`

> Also building **KAIROS for Teams** — a persistent memory layer for engineering teams so context never gets lost across Jira, GitHub, Slack, and docs. *(Early stage)*

---

### [Scratchers](https://github.com/Pankaj4152/Scratchers) — AI Systems Built From Scratch
Neural network in NumPy (MNIST), modular 6-stage RAG pipeline, custom vector store with cosine similarity, benchmarking framework for chunking strategies and LLMs.
`Python · NumPy · PyTorch`

---

### [Issuenix](https://issuenix.com) — Certificate Issuance & Verification Platform *(Live)*
Full-stack SaaS for issuing and verifying tamper-proof digital certificates. Bulk generation via CSV, automated SMTP delivery, hash-based forgery prevention, AI assistant (Gemini + RAG).
`FastAPI · PostgreSQL · React · Docker`
